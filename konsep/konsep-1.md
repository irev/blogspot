## $ ls (List files)
```
$ ls 
```

## ls Options

<table>
 <tr><td> -a </td> <td> Show all (including hidden)</td></tr>

-r

-t

-S

-l

-1
One file per line
-m
Comma-­sep­arated output
-Q
